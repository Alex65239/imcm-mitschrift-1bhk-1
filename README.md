## Playlist zu Funktionsweise des Internets

### Teil 1 - What is the Internet?

- das Internet wurde in den 1979er-Jahren entwickelt
- Motivation: Schaffung eines dezentralen Netzwerks dass auch nach einem Atomschlag noch funktioniert
(Kontext des Kalten Krieges)
- Funktionswiese: Paketvermittlung 
 (*Packet Switching*) - Nachrichten bzw. Datein werden in kleinere Pakete aufgeteilt und unabhängig voneinander über das Netzwerk verschickt
 - Internet: das Netz der Netze - besteht aus vielen kleineren Netzen unterschiedlicher Internetanbieter(*Internet Service Provider - ISP*, z.B. : Salzburg AG, Magenta, A1, ...)

 ### Teil 2 - The Internet: Wires, Cables and WiFi

 Informationen werden im Internet als Bits übertragen. Bits haben 2 Werte 0 und 1. 8 Bits zusammengefasst ergeben 1 Byte. Mit einem Bite kann man 256 verschiedene Werte speichern (2^8).

 Bits können über verschiedene Übertragungsmedien versendet werden. Die Anzahl der übertragenen Bits pro Sekunde wird als **Bandbreite** bezeichnet. Bei einer Bandbreite von 300MBit/sek können beispielsweise 300 Millionen Bite pro Sedunde über die Leitung laufen.
Übertragungsmedien können sein:

**1. Elektrizität / Kupferdraht (Ethernet)**

- billig
- einfach in der Verarbeitung
- weit verbreitet
- hohe Verluste über mittlere und lange Distanzen (hunderte Meter)

**2. Licht / Glasfaserkabel**

- schnelle Übertragung
- verlustfrei
- geeignet für Ozeankabel
- teuer und schwierig in der Verarbeitung 

**3. Funk / Radiowellen**

- hoher Komfort, Internet überall

### Teil 3 - The Internet IP-Addressen & DNS

- Protololle sind die Regeln des Kammunikation
- eines der wichtigsten Protokolle im Internet ist das Internet Protocol (IP)
- jedes Gerät im Internet hat zumindest eine (eindeutige) IP-Addresse, viele Geräte haben aber eine externe IP (ähnlich wie die Hausnummer) und eine interne IP (ähnlich wie die Raumnummer)
- das Domain Name System (DNS) übersetzt menschenlesbare Domainnamen (z.B. :www.google.com) in IP-Adrressen
- DNS-Server führen Tabellen mit Domainnamen und den entsprechenden Ip-Addressen
